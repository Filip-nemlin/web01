!DOCTYPE:

Används för att deklarera vilken version av HTML som används i dokumentet.

html:

Är rot- eller överordnat för hela HTML-dokumentet och omsluter allt annat innehåll på sidan.

head:

Innehåller metadata om dokumentet, som titel, teckenkodning och länkar till externa resurser.

title:

Används för att ange sidans titel, som visas i webbläsarens titelrad eller i bokmärken.

style:

Används för att definiera inlinestilregler för att styra sidans utseende.

link:

Används främst för att länka till externa stilmallar (CSS) och ikoner, som favicon.

body:

Omsluter den faktiska innehållet som ska visas på webbsidan.

p:

Representerar en avsnittsvis text och skapar en ny rad efter avsnittet.

headings (h1-h6):

Rubrikselement som används för att strukturera sidan och ge viktiga rubriker och underrubriker.

ol:

Används för att skapa en ordnad lista där varje objekt är numrerat. Varje objekt omsluts av ett li-element.

ul:

Används för att skapa en oordnad lista där objekten inte är numrerade. Varje objekt omsluts av ett li-element.

li:

Används inuti ol eller ul för att definiera enskilda objekt i en lista.

strong:

Används för att ge texten en stark betoning, vilket vanligtvis visas som fetstil.

em:

Används för att ge texten en betoning, vanligtvis visas som kursiv stil.

a:

Används för att skapa hyperlänkar till andra webbsidor eller resurser.

span:

Är ett inlinjelement som används för att applicera stil eller skapa en selektiv markering för en del av texten inom ett annat element.

main:

Används för att ange det huvudsakliga innehållet på sidan. Det ska finnas endast en main-element i ett dokument.

header:

Används för att definiera sidans rubrik, som oftast inkluderar logotyp, sidans titel och navigeringslänkar.

section:

Används för att gruppera innehåll i ett dokument i logiska sektioner. Det hjälper till med att organisera och strukturera sidan.

footer:

Används för att placera sidans avslutande innehåll, som copyright-information, kontaktuppgifter eller annat relevant innehåll.

article:

Används för att definiera ett självständigt innehåll som kan fungera som en fristående artikel eller nyhetsartikel inom en webbsida.

aside:

Används för att definiera innehåll som är relaterat till huvudinnehållet, men som kan ses som sekundärt eller sidoinnehåll.

div:

Är ett generiskt blockelement som används för att gruppera och strukturera innehåll för styling och layoutändamål. Det har ingen speciell semantisk betydelse.
